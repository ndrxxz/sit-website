# SIT Website

A modern and responsive IT services website designed to showcase various technology solutions offered by the company. The website features a clean UI with smooth navigation, categorized services, and an interactive landing page.

---

## 📌 Features

- Modern and responsive design
- Landing page with company introduction
- Interactive navigation
- Call-to-action buttons
- Service details page
- Clean and minimal user interface

---

## 🏠 Home Page

The homepage introduces the company and highlights its core services.

### Hero Section

- Company tagline
- Short company description
- **Book a Call** button
- Embedded promotional/company video

### Navigation

- Home
- Services
- About Us
- Catalogue

---
